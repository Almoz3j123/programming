أهلاً بيك في كورس بايثون الشامل

يا باشا، لو عمرك ما برمجت قبل كده أو لو عندك خلفية بسيطة وعايز تتعلم بجد، الكورس ده ليك. في الكورس ده هنتعلم كل حاجة عن لغة بايثون: من الأساسيات لحد الاحتراف. مش بس كده، كمان هتعرف إزاي تكتب سكربتات وتشغلها وتتعامل مع المشاريع زي المحترفين.

يعني إيه بايثون؟ وليه أتعلمها؟

بايثون يا نجم هي لغة برمجة سهلة أوي ومشهورة جدًا. ليه؟ لأنها بتستخدم في كل حاجة تقريبًا:

  • برمجة المواقع.
  • الذكاء الاصطناعي (AI) والتعلم الآلي (Machine Learning).
  • تحليل البيانات.
  • تصميم الألعاب.
  • كتابة سكربتات أوتوماتيكية.

وبتتميز إنها سهلة القراءة والكتابة، فمش هتحتاج تكون عبقري علشان تبدأ.

هنتعلم إيه في الكورس ده؟

  1. أساسيات اللغة: إزاي تكتب أول برنامج ليك؟ يعني مثلاً برنامج "Hello, World!".
  2. المتغيرات: إزاي تخزن البيانات وتستخدمها.
  3. الحلقات (Loops): لو عايز تكرر حاجة أكتر من مرة.
  4. الدوال (Functions): إزاي تكتب أكواد منظمة وسهلة التعديل.
  5. التعامل مع الملفات: إزاي تقرأ وتكتب ملفات باستخدام بايثون.
  6. المكتبات: زي Numpy وPandas وMatplotlib لتحليل البيانات.
  7. مشروع كامل: هنكتب سكربت بايثون يحل مشكلة معينة وهنشغله.

أول خطوة: شغل بايثون على جهازك

علشان تبدأ، لازم تنزل بايثون على جهازك:

  1. روح لموقع python.org.
  2. نزل النسخة المناسبة لنظام التشغيل بتاعك (Windows، Mac، Linux).
  3. بعد التنصيب، افتح الـ Terminal أو Command Prompt واكتب:
    python --version

    لو ظهرت النسخة يبقى كله تمام!

مشروع صغير: برنامج آلة حاسبة

هنكتب برنامج آلة حاسبة بسيط جداً:

def calculator():
  num1 = float(input("أدخل الرقم الأول: "))
  operation = input("أدخل العملية (+, -, *, /): ")
  num2 = float(input("أدخل الرقم الثاني: "))
  if operation == "+":
    print("النتيجة:", num1 + num2)
  elif operation == "-":
    print("النتيجة:", num1 - num2)
  elif operation == "*":
    print("النتيجة:", num1 * num2)
  elif operation == "/":
    print("النتيجة:", num1 / num2)
  else:
    print("عملية غير صحيحة")

calculator()
