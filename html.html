أهلاً بيك في كورس HTML الشامل

في الكورس ده، هنتعلم مع بعض لغة HTML اللي هي الأساس اللي بُنيت عليه جميع المواقع على الإنترنت. لو كنت مبتدئ أو لو حابب تتعلم من البداية لحد الاحتراف، فإنت في المكان الصح. هنشرح كل شيء بالتفصيل، بداية من الأساسيات لحد المتقدم، عشان تقدر تبني مواقعك الخاصة بكل سهولة.

HTML (HyperText Markup Language) هي لغة الترميز الخاصة بالمحتوى اللي بيظهر على الإنترنت. لما بنقول "موقع ويب"، بنقصد مجموعة من الصفحات اللي تم كتابتها باستخدام HTML. اللغة دي مش لغة برمجة بمعنى الكلمة زي بايثون أو جافا، لكنها لغة هيكلية بتحدد شكل وموضع المحتوى على الصفحة، وبيتم دمجها مع CSS (اللي بتحدد شكل وتصميم الصفحة) وجافا سكريبت (اللي بتضيف التفاعلية) علشان تحصل على موقع ويب كامل ومتفاعل.

إيه هي HTML؟

الـ HTML بتستخدم أكواد وعناصر (Tags) علشان تحدد هيكل الصفحة. مثلاً، لو كنت عايز تضيف عنوان أو نص أو حتى صورة أو رابط، هتستخدم الأكواد الخاصة بكل عنصر. العناصر دي هي عبارة عن وسوم مكتوبة بين إشارات الزاوية (مثل <div> و<h1>). داخل الصفحة، كل عنصر له دور محدد بيؤثر على شكل الموقع وطريقة عرضه.

مثال بسيط، لو عايز تضيف عنوان رئيسي لصفحة الويب، هتستخدم وسم <h1> زي كده:

<h1>مرحباً بك في تعلم HTML!</h1>

النتيجة هتكون أن النص "مرحباً بك في تعلم HTML!" هيظهر في الصفحة بشكل بارز كعنوان رئيسي.

أهمية HTML في تطوير الويب

الـ HTML هي أساس بناء المواقع على الإنترنت. كل صفحة ويب بتبدأ باستخدام HTML علشان نحدد هيكل الصفحة. من غير HTML، ما نقدرش نبني أي موقع. وده ليه؟ لأن HTML هي اللي بتحدد الهيكل الأساسي اللي هيتضاف عليه كل شيء تاني زي النصوص، الصور، الفيديوهات، الروابط، والأزرار التفاعلية.

لكن طبعاً HTML مش لوحدها كفاية علشان يكون عندك موقع ويب رائع. لازم تتعلم بعد كده CSS علشان تقدر تضيف تصاميم وألوان وتنسيقات جميلة للصفحة، ثم بعد كده تتعلم JavaScript عشان تضيف تفاعلية وتحكم ديناميكي في الصفحة زي الأزرار اللي بتنقر عليها لفتح روابط أو عرض محتوى معين. HTML هي البداية، بس لازم تواصل التعلم وتضيف المهارات التانية.

المحتويات الأساسية في HTML

في الـ HTML، كل شيء بيتم بناؤه باستخدام العناصر (Elements) أو الوسوم (Tags). كل عنصر ليه وظيفة معينة:

  • <h1> لـ <h6>: لتحديد العناوين من الأكبر للأصغر.
  • <p>: لإضافة الفقرات النصية.
  • <a>: لإضافة الروابط.
  • <img>: لإضافة الصور.
  • <ul> و <ol>: لإضافة القوائم المرتبة وغير المرتبة.
  • <div>: لتقسيم الصفحة إلى أقسام.

كل عنصر بيحتوي على خصائص (Attributes) بتحدد كيف هيتصرف العنصر ده. مثلاً، وسم <a> اللي بنستخدمه علشان نضيف روابط بيحتوي على خاصية اسمها "href" وهي اللي بتحدد رابط الصفحة اللي هيتنقل ليها المستخدم عند الضغط على الرابط.

مثال على إضافة رابط:

<a href="https://example.com">اضغط هنا للذهاب للموقع</a>

الضغط على الرابط ده هيوجهك مباشرة إلى الموقع "example.com". وده بيبين قد إيه HTML بسيطة وسهلة في الاستخدام.

دور الـ CSS و JavaScript مع HTML

بعد ما تكتب أكواد HTML لإنشاء هيكل الموقع، هتحتاج تضيف له تصميم علشان يكون الموقع شكله جميل ومناسب. هنا بيجي دور الـ CSS، وهي لغة خاصة بتنسيق تصميم المواقع. الـ CSS بتتحكم في الألوان، الخطوط، المسافات، والتنسيقات بشكل عام. بدون CSS، الموقع هيفضل شكل الصفحة النصية البسيطة، واللي مش هتكون جذابة.

أما الـ JavaScript فهي اللغة اللي بتضيف التفاعلية للموقع. مثلاً، لو عندك صفحة فيها زر بيخفي أو يظهر بعض المحتوى عند الضغط عليه، ده بيتم باستخدام JavaScript. زي ما هنتعلم في المستقبل، الـ HTML هي الهيكل، الـ CSS هو التصميم، وJavaScript هي التفاعل.
